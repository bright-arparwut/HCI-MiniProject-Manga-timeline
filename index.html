ภาพรวมด้านบน visualize เกี่ยวกับช่วงเวลาของการตีพิมพ์การ์ตูนญี่ปุ่น (มังงะ) ทุกเรื่องที่อยู่ใน data set ซึ่งแสดงให้เห็นถึงความหลากหลายในการเล่าเรื่องที่พัฒนาขึ้นตลอดช่วงหลายปีที่ผ่านมา แต่ละแท่งบนเส้นเวลาแสดงถึงซีรีส์มังงะเฉพาะตัว โดยความยาวของแท่งแสดงช่วงเวลาตั้งแต่เริ่มต้นของแต่ละซีรีส์จนถึงจุดจบหรือสถานะปัจจุบันหากยังคงดำเนินอยู่ การ visualizeในรูปแบบนี้ช่วยให้สามารถเปรียบเทียบระยะเวลาของแต่ละซีรีส์ได้อย่างรวดเร็ว ทำให้สามารถเข้าถึงข้อมูลเกี่ยวกับอายุของซีรีส์ รวมถึงความนิยมและอิทธิพลที่มีต่อผู้อ่านอีกทางหนึ่ง

การ Interactive ผ่าน Tooltip จะเผยให้เห็นข้อมูลเชิงลึกที่ลึกซึ้งยิ่งขึ้นในแต่ละซีรีส์ โดยการวางเมาส์เหนือแท่งจะแสดงข้อมูลโดยละเอียด รวมถึงชื่อซีรีส์ ผู้แต่ง ผู้จัดพิมพ์ กลุ่มเป้าหมาย จำนวนเล่มที่ตีพิมพ์ และยอดขาย รายละเอียดในส่วนนี้นี้ช่วยเสริมความเข้าใจของผู้ชม นำเสนอมุมมองเกี่ยวกับภาพรวมของซีรีส์และอิทธิพลทางวัฒนธรรม การโต้ตอบดังกล่าวเชื้อเชิญให้มีการสำรวจอย่างลึกซึ้งมากขึ้น ทำให้ผู้ชมสามารถค้นพบคู่มือใหม่ๆ และกลับไปดูคู่มือที่คุ้นเคยด้วยมุมมองใหม่
